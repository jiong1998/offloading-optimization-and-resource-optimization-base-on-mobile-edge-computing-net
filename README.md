# offloading-optimization-and-resource-optimization-base-on-mobile-edge-computing-net
基于边缘计算网络的卸载优化及资源优化

考虑了一个多用户单窃听者的移动边缘计算(MEC)网络，通过使用深度强化学习算法，并结合凸优化，对MEC网络进行了卸载优化与资源优化。
